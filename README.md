# ZyveraWelcomeMessage

[![Paper](https://img.shields.io/badge/Paper-1.16%2B-blue)](https://papermc.io)
[![Folia](https://img.shields.io/badge/Folia-supported-green)](https://github.com/PaperMC/Folia)
[![Java](https://img.shields.io/badge/Java-17-orange)](https://adoptium.net)
[![Maven](https://img.shields.io/badge/Build-Maven-red)](https://maven.apache.org)

Advanced Minecraft Welcome Message Plugin mit **Farbverlauf-Support** (Gradients), Feuerwerk und Sounds.

---
## Features

- 🎨 **Farbverläufe** via MiniMessage (`<gradient:#farbe1:#farbe2>`)
- 🌍 **Globale Nachricht** für alle Spieler
- 👤 **Individuelle Nachrichten** pro Spieler (Name oder UUID)
- 🎆 **Feuerwerk-Effekte** vollständig konfigurierbar
- 🔊 **Sound-Effekte** (nur für Spieler oder alle)
- 🥇 **Erst-Join Nachricht** (erkennt neue Spieler)
- ⚡ **Folia-kompatibel** (RegionScheduler)
- 🔄 **/zwm reload** – kein Serverneustart nötig
- 👁️ **/zwm preview** – Nachrichten im Spiel vorschauen
